FOLDER STREAM - Taruh File Texture Bendera Di Sini
====================================================

Taruh file-file texture bendera Indonesia di folder ini dengan nama:

1. prop_flag_us.ytd (atau .png)
2. prop_flag_us_s.ytd (atau .png)
3. prop_flag_usboat.ytd (atau .png)

Format yang didukung:
- .ytd (GTA V Texture Dictionary)
- .png (akan di-convert otomatis)
- .dds (DirectDraw Surface)

Ukuran yang disarankan:
- 1024x512 pixels (tinggi x lebar)
- Atau 512x256 pixels

Cara termudah:
1. Buat gambar PNG dengan 2 warna:
   - Atas: Merah #FF0000
   - Bawah: Putih #FFFFFF
   
2. Save dengan nama di atas

3. Restart resource: restart indonesia_flag

ATAU download texture yang sudah jadi dari komunitas FiveM!
